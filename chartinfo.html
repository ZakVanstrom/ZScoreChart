Free Interactive Z-Score Chart on your site!

Basic Info:

The chart on the homepage was created using Plotly and Python, was exported to HTML, and can be easily inserted anywhere on the web.

The values were generated using a brute-force algorithm hitting on each of 800 data points (Z-Scores: [-4 , 4], Step = 0.01) accurate to 6 decimals. It wasn't efficient, but it worked better for me than implementing an algorithm for the Gaussian Integral. 

The whole thing is open-source and the repository is hosted on GitHub.

Please feel free to use it and change it however you'd like.
That's what I made it for  :)

Simple Install:

  1. Download the zscoresChart.html file from the GitHub repo: Z-Score Project GitHub
  2. Add zscoresChart.html to an accessible folder in your webpage's files
  3. Add a link wherever you'd like the chart to show up inside an HTML file
  4. that's it. Run your site, and it should show up

I had a couple reasons in mind to finish this project:

Meaning that nearly all of the interactive Z-Score charts soon won't be usable

I'm striving to help educators however I can

Teaching statistics isn't easy and understanding Z-Scores for the first time can be a special pain. This interactive chart might help students get a feel for Z-Scores themselves.
